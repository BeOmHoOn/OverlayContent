># OverlayContent
># 오버레이 컨텐트 컨트롤
>Overlay Text/Content FrameworkElement for WPF<br/>
>WPF사용자를 위한 오버레이 컨텐트 컨트롤입니다.

>## 예제 이미지

>## Properties
>## 속성
>#### Content : Text / Path
>Value of text or stream geometry which show.
>표시될 텍스트나 스트림 지오메트리입니다.
>
>#### Scale : Double
>Content scale property
>컨텐트의 스케일입니다. 기본값은 1이며 배로 계산합니다.
>
>#### Fontsize : Double
>If Content is Text, that font size.
>컨텐트가 텍스트일 경우 폰트사이즈를 지정할 수 있습니다.
>
>#### Fill : Color
>Color of rectangle
>사각형의 색입니다.
>
>#### FontFamily : FontFamily
>Text's font family
>텍스트일 경우 폰트를 지정할 수 있습니다.
>
>#### RadiusX : Double
>Control's corner radius X
>사각형 모서리의 좌우 반지름입니다.
>
>#### RadiusY : Double
>Control's corner radius Y
>사각형 모서리의 상하 반지름입니다.
>
>#### DebugMode : Bool
>When True, draw some lines for check position
>True일 경우 위치를 확인하기 위한 선을 그려줍니다.
